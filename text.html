姚期智院士

个人简介

姚期智院士是中国著名的计算机科学家和密码学家,他于1946年出生于湖北省孝感市。他的研究方向包括计算理论及其在密码学和量子计算中的应用,在三大方面具有突出贡献：

  • 创建理论计算机科学的重要次领域：通讯复杂性和伪随机数生成计算理论；
  • 奠定现代密码学基础，在基于复杂性的密码学和安全形式化方法方面有根本性贡献；
  • 解决线路复杂性、计算几何、数据结构及量子计算等领域的开放性问题并建立全新典范。

经典的非对称加密算法——RSA算法

算法原理

  • 选择两个大质数p和q,并计算它们的乘积n=p*q;
  • 计算欧拉函数φ(n)=(p-1)*(q-1);
  • 选择一个整数e(1< e < φ(n))作为公钥,使得e与φ(n)互质;
  • 计算d使得d*e ≡ 1 (mod φ(n)),即d是e在模φ(n)意义下的逆元素,即为私钥;
  • 公钥为(n,e),私钥为(n,d);
  • 加密时,将明文m用公钥加密得到密文c,即c ≡ m^e (mod n);
  • 解密时,用私钥对密文进行解密,即m ≡ c^d (mod n)。

RSA算法的安全性基于大质数分解的困难性。由于目前没有有效的算法可以在多项式时间内分解大质数,因此RSA算法被认为是一种安全可靠的加密算法。

百万富翁问题

解决方案

假设小明和小美分别得到公司的一笔年终奖,分别为x和y(具体数值不公开),他们想要比较谁的年终奖更多,但不想直接透露自己的年终奖数额。这时候,小明可以生成一对RSA算法中的公钥和私钥,并将公钥公开给小美。

  • 小美选择一个随机数k,并用小明提供的公钥对其进行加密得到c = k^e (mod n),其中e是小明提供的公钥中的指数,n是两个大质数p和q相乘得到的结果;
  • 小美将加密后的结果c - x发送给小明;
  • 小明收到c - x后,使用自己持有的私钥计算n个数z_i = (c - x + i)^d (mod n),其中d是小明的私钥,1 <= i <= n;
  • 小明将y + 1 <= i <= n的各个数加一,并将所有z_i发送给小美;
  • 小美收到一系列数后,检查第x个数是否等于k。若等于,则年终奖x <= y;不等于,则年终奖x > y。
